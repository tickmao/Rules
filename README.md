# Rules

一个多客户端代理规则仓库，统一维护以下配置：

- Surge
- Loon
- QuantumultX
- Clash Meta
- Shadowrocket

## 目录

```text
.
├── Surge/
├── Loon/
├── QuantumultX/
├── Clash/Meta/
├── Shadowrocket/
├── scripts/
└── .github/workflows/
```

- Surge: `https://raw.githubusercontent.com/tickmao/Rules/master/Surge/Surge.conf`
- Loon: `https://raw.githubusercontent.com/tickmao/Rules/master/Loon/Loon.conf`
